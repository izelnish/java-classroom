# Java Classroom (Migrated to PHP/HTML)

This project has been migrated from a Java Spring Boot backend to a PHP backend for hosting on InfinityFree.

## Project Structure

- **`index.html`**: The main frontend application (Single Page App).
- **`php/`**: Contains the active PHP backend scripts (`login.php`, `signup.php`, `config.php`, `setup_db.php`).
- **`legacy_java/`**: Archive of the original Java Spring Boot source code and Maven configuration.
- **`.vscode/`**: VS Code configuration files.

## Hosting Info

- **Frontend**: Hosted at `ftpupload.net/htdocs/index.html`
- **Backend**: Hosted at `ftpupload.net/htdocs/php/`
- **Database**: MySQL on InfinityFree (`sql206.infinityfree.com`)

## Key Files

- `index.html`: Contains all UI, Chatbot logic (Gemini AI), and API calls.
- `php/config.php`: Database connection settings.
